Access the z-axis mapping tool
zaxis.uvic.ca

Basic Features

  • Text selection
  • Place selection
  • Map selection
  • Sentiment Mapping
  • Temporal Mapping

Advanced Features: custom maps

  • Download a custom map from mapwarper.net
  • Export the downloaded .png file as a .jpg file
  • Open the KML file in a text editor
  • Import the .jpg file and copy-paste the borders (NSEW) from the KML file
  • Give the map a title, add it to list, and select it in the MAP view

Advanced Features: custom word banks

  • Download the example word bank
  • Edit the file and replace the example words with a list of your choosing
  • Give your word bank a title and add the word bank to list
  • In the MAP view, you may now select your wordbank as a sentiment
  • Wordbanks need not reflect sentiment alone, they may list thematic concerns (i.e. technology), parts of speech, or something else altogether
